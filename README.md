# Sequora

Marketing site for **Sequora** — a climate-intelligence platform and ESG consultancy that helps teams measure their footprint, reduce emissions, and back verified restoration projects.

Clean, hand-coded build. No page builders.

## Stack

- **Next.js** (App Router) + **TypeScript**
- Custom CSS design system (forest-green / mint / warm-cream palette)
- **Poppins** + **Host Grotesk** via `next/font` (self-hosted)
- Scroll reveals, count-ups, hero marquee, testimonial slider and FAQ accordion — all dependency-free
- Static OG image + custom favicon

## Pages

Home · About · Services · Pricing · Blog · Contact

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build
npm run start
```

## Assets

Images in `public/img` are temporary placeholders to be swapped for final brand assets.
